# Enchanted Crystals Mod

A magical Minecraft mod that adds enchanted crystals and magical items!

## Features

✨ **Enchanted Crystal Ore** - Find and mine magical crystals in the world
⚡ **Crystal Staff** - A powerful weapon with teleportation abilities (Shift+Click to teleport forward 10 blocks)
💎 **Magical Shards** - Crafting materials for magical items
🔮 **Enchanted Crystal Block** - A decorative block made from pure magic

## Installation

### What You Need
- **Minecraft 1.20.1**
- **Forge Loader** (version 47.2.0 or compatible)

### How to Install

1. **Download Forge Installer** from https://files.minecraftforge.net/ (version 1.20.1)
2. **Run the installer** and select "Install client"
3. **Download the mod JAR** from the Releases section of this repo
4. **Open your Minecraft folder**:
   - Windows: `%appdata%\.minecraft`
   - Mac: `~/Library/Application Support/minecraft`
   - Linux: `~/.minecraft`
5. **Open the `mods` folder** (create it if it doesn't exist)
6. **Drag the mod JAR file** into the `mods` folder
7. **Launch Minecraft** with the Forge profile
8. **Enjoy!** The mod items and blocks should now appear in the creative menu

## Usage

### Finding Materials
- Mine **Enchanted Crystal Ore** with a stone pickaxe or better
- Collect **Magical Shards** and **Enchanted Crystals**

### Crystal Staff
- Use it as a powerful sword
- **Hold Shift + Right Click** to teleport forward 10 blocks
- Each teleport uses durability

## Building from Source

Requirements:
- Java 17 or higher
- Gradle

```bash
# Clone the repo
git clone https://github.com/lecroissant00/EnchantedCrystals.git
cd EnchantedCrystals

# Build the mod
gradlew build

# Find the JAR in build/libs/
```
